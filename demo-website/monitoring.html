📱 How to Setup Android Phone as Camera

  1. Install IP Webcam: Download "IP Webcam" from Google Play Store
  2. Connect to WiFi: Ensure phone & PC are on the same WiFi network
  3. Start Server: Open app → Scroll down → Click "Start server"
  4. Note IP Address: App shows URL like http://192.168.1.100:8080
  5. Enter in Form: Copy just the IP part: 192.168.1.100:8080
  6. Test Connection: Click "Test Camera" button

💡 Tips:

  • Keep phone plugged in to avoid battery drain
  • Use phone stand/tripod for stable positioning
  • Adjust video quality in app settings if needed
  • You can use multiple phones as different cameras
